# 📉 BTC Trading Analysis & Live PnL Dashboard

歡迎來到我的比特幣交易分析專案！這個專案結合了 **即時行情分析報告** 與 **真實帳戶回測儀表板**，旨在提供 100% 透明且可驗證的交易策略紀錄。

---

## 🚀 核心功能與特色

### 1. 真實帳戶回測 (PnL Dashboard)
串接幣安期貨 (Binance Futures) API 實時顯示帳戶績效，絕無造假：
- **多維度區間切換**：支援 7天 / 30天 / 3個月 / 全時段 的獲利統計。
- **精確投報率 (ROI)**：系統自動識別資金轉入/轉出 (Transfer) 與 交易盈虧 (Trading PnL)，確保報酬率計算精確到小數點後兩位，排除任何非交易因素。
- **視覺化走勢**：透過累積獲利率曲線 (AreaChart) 與單日淨利直條圖 (BarChart) 掌握每一天的績效波動。

### 2. 專業分析報告 (Daily Reports)
每一份報告都是我們針對當前 BTC 市場結構、價格行為 (Price Action) 所做的深度解讀：
- **置頂「關於我」**：引導新讀者快速了解本站的運作原理與策略背景。
- **折疊式側邊導覽**：直覺切換歷史分析，所有過去的建議與後續驗證均公開可供檢視。

### 3. 個人化閱讀體驗
- **極簡美學**：專為交易者設計的 UI，簡潔且無干擾。
- **深色模式 (Dark Mode)**：針對夜間盯盤需求，隨時切換舒適的視覺風格。
- **響應式排版**：無論在手機還是電腦上，都能流暢閱讀程式碼與圖表。

---

## 🛠️ 專案技術架構 (For Technical Users)

本專案採前後端分離架構部署：
- **Frontend**: Vite + React + Recharts (資料可視化) + CSS Variables (動態主題)。
- **Backend (API)**: Node.js + Express，具備高效能快取與 Binance 簽署認證機制。
- **Security**: 所有敏感 API Key 已於伺服器端隔離，確保讀者隱私與資產安全。
